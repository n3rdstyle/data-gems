How to Use Data Gems

Step 1: Build Your Profile

  1. Click the Data Gems icon in your browser toolbar
  2. Option A: Take the interactive "Quiz of Gems" for a quick profile setup
  3. Option B: Manually add information using the "Add New Information" button
  4. Organize your information into categories like Work, Preferences, Hobbies, etc.
  5. OPTIONAL: Create individual subprofiles for different contexts (e.g., travel, food or favorite brands)
üéØ NEW in v1.1.0: The Quiz of Gems provides a fun, interactive way to quickly build a comprehensive profile through guided questions!

Step 2: Inject Your Profile

  1. Visit any supported AI platform (ChatGPT, Claude, Gemini, Perplexity)
  2. Look for the colorful "Inject my profile" button near the chat input
  3. Click it and select what to inject (full profile or a specific subprofile)

Plus: There is an auto-injection option. See the Advanced Features section below.

Step 3: Personal Description (Optional)

Add a custom description that appears with your profile:

  1. Set your name in the profile popup
  2. Write a brief personal description
  3. This helps AI getting a brief intro about you, before even analyzing all your personal information

Step 4: Upload Profile Image (Optional)

  1. Click the avatar area in the profile popup
  2. Upload a profile image

Note: The image is only for visual display in the extension - it is not sent to AI platforms

Advanced Features

Subprofiles

Create focused context sets for different situations, for example:

  • Travel Planning: Preferred destinations, travel style, budget preferences
  • Food & Dining: Dietary restrictions, favorite cuisines, cooking preferences
  • Shopping: Favorite brands, style preferences, budget ranges
  • Entertainment: Movie genres, music tastes, hobbies and interests

How to Create and Use Subprofiles

  1. Create: Click the "+" button next to "Active Profile" in the extension popup
  2. Name: Give your subprofile a descriptive name (e.g., "Travel Planning")
  3. Add Information: Add relevant context items to this subprofile
  4. Select: Use the dropdown menu to switch between subprofiles
  5. Inject: When you click "Inject my profile", you can choose to inject the active subprofile or your full profile

Prompt Library

Create and manage a library of reusable prompts that can be quickly injected into AI conversations.

Using the /prompt Command

Type the command directly in the AI chat input to inject a specific prompt by its tag:

/prompt:work # Injects the prompt tagged with "work"
/prompt:coding # Injects the prompt tagged with "coding"
/prompt:travel # Injects the prompt tagged with "travel"

How to Set Up Prompts

  1. Create prompts in the settings popup (under "Beta Lab")
  2. Assign a unique tag to each prompt (e.g., work, coding, travel)
  3. Use the /prompt:tag command to quickly inject any saved prompt

Auto-Injection

Configure Data Gems to automatically inject your profile:

  1. Open settings popup (under "Beta Lab")
  2. Enable "Auto-inject on page load"
  3. Set delay time (default: 2 seconds)
  4. The currently selected profile/subprofile in the extension will be automatically injected

Import/Export

Backup or transfer your data between devices:

  1. Click the Export button to download your data as JSON
  2. Save the file in a secure location
  3. On another device, click Import and select your backup file
  4. Your profile will be restored with all settings intact
